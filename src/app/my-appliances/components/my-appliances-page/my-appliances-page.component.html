My Appliances

This page is a list of active appliances you have launched.

Each log entry keeps key appliance and instance information, also giving you a link to access a web interface that the given appliance may offer.
From this page you may refresh the status of an appliance. You can also delete and archive an appliance. Please note that deleting an appliance is a destructive, irreversible action. If you wish to preserve the state of your appliance please make sure to back it up first.
If an appliance has been shut down or is otherwise no longer available you can remove it from this list by archiving it.
All appliances you've launched, including archived appliances, are listed on the Launch History page.
